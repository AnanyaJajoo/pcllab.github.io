# Lab Website - MkDocs

To run MkDocs locally: 
- cd to this folder
- Run `mkdocs serve` or `python3 -m mkdocs serve`
- View the site at `http://127.0.0.1:8000/`

To stop the local server:
- Press CTRL-c

To build the site:
- Run `mkdocs build` or `python3 -m mkdocs build`
